# 部署环境变量示例

> 本文件只放占位示例，不写真实密码、数据库地址和密钥。真实值以宝塔服务器环境变量或 `backend/env/.env.prod` 为准。

## 业务库

```env
DATABASE_TYPE="mysql"
DATABASE_HOST="127.0.0.1"
DATABASE_PORT=3306
DATABASE_USER="smartqa_user"
DATABASE_PASSWORD="change_me"
DATABASE_NAME="smartqa"
```

## Redis

```env
REDIS_ENABLE=True
REDIS_HOST="127.0.0.1"
REDIS_PORT=6379
REDIS_PASSWORD="change_me"
REDIS_DB_NAME=1
```
