## 3. 首次依赖安装

本机依赖我已经帮你检查并安装过。如果以后换电脑、重拉仓库或删除依赖目录，按下面命令安装。

### 后端依赖

```powershell
cd F:\Project\SmartQAServiceCloud\backend
python -m pip install -r requirements.txt
```

### 前端依赖

```powershell
cd F:\Project\SmartQAServiceCloud\frontend\web
corepack pnpm install --frozen-lockfile
```

当前本机环境：

- Python：3.13.5
- Node.js：24.15.0
- pnpm：9.15.3

## 4. 启动后端

打开一个 PowerShell 窗口：

```powershell
cd F:\Project\SmartQAServiceCloud\backend
$env:PYTHONIOENCODING="utf-8"
python main.py run --env dev
```

看到服务地址 `http://localhost:8001`，并且日志出现 `Application startup complete` 即启动成功。

后端本地 Redis 当前关闭，系统会走内存降级；生产部署到宝塔服务器时建议启用正式 Redis。

## 5. 启动前端

再打开一个 PowerShell 窗口：

```powershell
cd F:\Project\SmartQAServiceCloud\frontend\web
corepack pnpm run dev
```

看到 Vite 输出 `http://localhost:5180/web` 后，用浏览器打开该地址登录。

## 7. 常用维护命令

查看数据量：

```powershell

```


创建质检任务：

```powershell
cd F:\Project\SmartQAServiceCloud\backend
python scripts\smartqa_pipeline.py create-tasks --limit 10
```

调用阿里模型执行质检：

```powershell
cd F:\Project\SmartQAServiceCloud\backend
python scripts\smartqa_pipeline.py run-ai --limit 1
```

## 9. 停止服务

如果是按第 4、5 节在 PowerShell 窗口里启动的，直接在对应窗口按 `Ctrl + C`。

如果是后台进程方式启动的，可以查看端口对应进程：

```powershell
Get-NetTCPConnection -State Listen | Where-Object { $_.LocalPort -in 8001,5180 } | Select-Object LocalAddress,LocalPort,OwningProcess
```

停止指定进程：

```powershell
Stop-Process -Id 进程ID
```
